Proyecto Easy MERN Stack Server

Este proyecto es un servidor de lado del servidor construido con Node.js, Express y MongoDB. Está diseñado para proporcionar una API para aplicaciones web y móviles.

Información del Proyecto

  • Nombre del Proyecto: easy-mern-stack-server
  • Versión: 1.0.0
  • Autor: Miles Till
  • Licencia: MIT
  • Palabras Clave: server, api, node, express, mongodb

Dependencias

Dependencias Principales:

  • body-parser: ^1.18.3 - Middleware para analizar datos en solicitudes HTTP.
  • easy-mern-stack-shared: ^1.0.0 - Compartir código y recursos en el proyecto.
  • express: 4.17.1 - Framework de aplicaciones web para Node.js.
  • http-status-codes: ^1.3.0 - Colección de códigos de estado HTTP y mensajes asociados.
  • mongoose: 5.5.13 - Biblioteca de modelado de objetos MongoDB para Node.js.

Dependencias de Desarrollo:

  • monopack: ^0.2.6 - Herramienta de desarrollo y construcción.

Scripts

  • build: rimraf build && yarn monopack build src/main.js
  • dev: nodemon --inspect src/main.js
  • start: yarn dev
